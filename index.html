Privacy Policy for Group Hopper

Last updated: December 6, 2024

Overview

Group Hopper is a Chrome extension that helps manage tab groups. We do not collect, store, or transmit any personal information.

Data Usage

This extension only uses the following Chrome permissions:

  • tabs: Required to access and modify browser tabs
  • tabGroups: Required to manage tab groups
  • commands: Required for keyboard shortcuts
  • scripting: Required to interact with browser tabs

Data Storage

All operations are performed locally within your browser. No data is sent to external servers.
